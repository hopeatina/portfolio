# Portfolio social film

`Continuity Signal` is a 28-second, 1080×1920 Remotion film built from the redesigned portfolio, Hope's existing portrait, and an original in-repo score.

## Commands

- `npm run video:studio` — open the Remotion composition
- `npm run video:poster` — render the social cover
- `npm run video:render` — render the H.264 visual/audio master
- `npm run video:final` — render and create the true-peak-safe, social-loudness final export

The post-render share master is `video/output/hope-atina-portfolio-social-final.mp4`. It includes a true-peak-safe loudness pass measured at −14.1 LUFS and −0.9 dBFS true peak.

Pre-production evidence lives in `video/preproduction/`; frame and media conformance evidence lives in `video/qa/`.
