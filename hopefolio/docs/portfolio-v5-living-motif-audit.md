# Portfolio V5 — living motif audit

## Reference surface

The V4 portfolio itself is the brand source of truth. The five baseline captures in
`docs/verification/v5-baseline` cover the homepage, work index, OrgX case study, About,
and Hiring surfaces. The homepage hero is the primary reference.

## Motif deconstruction

| Element | Meaning | Behavior |
| --- | --- | --- |
| Woven field | Many actors and systems | Drifts with the viewer and resolves into a trace |
| Chartreuse signal | Continuity | Travels across sections and architecture layers |
| Heat / cold registration | Human judgment and client handoff | Separates at a boundary, rejoins at proof |
| Aperture | Inspection | Follows intent and reveals evidence in place |
| Branch | Specialist execution | Forks into surfaces without losing the source context |
| Repaired node | Decision / receipt | Marks where an outcome returns as organizational memory |

The brave choice is a single continuous signal that changes topology with the content.
It is not a repeated brand background: each form explains the idea beside it.

## Remediation contract

1. Carry a living signal through every primary page and change its form semantically.
2. Make inspection spatial and responsive, with reduced-motion fallbacks.
3. Expose rationale before stack names: pressure → decision → reason → consequence.
4. Show OrgX as a system of web, MCP, plugin, widget, agent, artifact, quality, and proof surfaces.
5. Rebuild About as a causal story: roots → rigor → consequence → collaboration → founder thesis.
6. Re-score every pass after desktop and 375px browser verification; no pass may remain below 9.
